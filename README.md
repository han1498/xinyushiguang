# 心语拾光 · Lily Whisper

> AI 智能信息管理助手
>
> 一个利用 AI Coding 开发的微信小程序，尝试将 AI 对话、长期记忆、待办与日程管理结合，让聊天中的重要信息能够沉淀为可管理的数据。

---

## 项目简介

日常生活和工作中，很多重要的信息其实都来自聊天。

例如：

- "明天下午三点开会"
- "月底记得提交材料"
- "我喜欢喝拿铁"

这些内容往往会随着聊天记录不断增加而被遗忘，需要再切换到待办、日历或备忘录重新整理。

因此，我尝试利用 AI Coding 开发一款 AI 智能信息管理助手，希望让 AI 不只是聊天，而是能够帮助用户自动整理聊天中的重要信息，提高个人信息管理效率。

---

## 主要功能

### 💬 AI 对话
- 基于 DeepSeek API 的自然语言对话
- 支持自定义 AI 人设、身份和系统提示词
- Markdown 富文本回复
- 情绪识别与状态反馈

### 🧠 长期记忆

AI 自动提取聊天中的用户偏好、习惯和重要事件，形成长期记忆。

例如：

- 喜欢喝咖啡
- 最近准备找工作
- 喜欢旅行

这些内容会参与后续对话，使回复更加连贯。

### ✅ 待办事项
- 日历视图
- 每日待办管理
- 完成状态记录
- 支持长期/自定义有效期

### 📅 日程与提醒

AI 自动识别聊天中的时间信息。

例如：

- 明天下午三点开会

可整理为日程或提醒事项，方便后续查看和管理。

### 📝 智能备忘

统一管理：

- 用户记忆
- 日程事件
- 提醒事项
- 用户资料

方便用户查看、编辑和维护 AI 提取的数据。

## 项目特点

相比普通 AI 聊天，我更希望探索 AI 在个人信息管理场景中的应用。

整个流程为：

```
聊天
  ↓
AI 理解
  ↓
提取有效信息
  ↓
整理为：长期记忆 / 待办事项 / 日程提醒
  ↓
后续持续调用
```

让聊天中的信息真正发挥价值，而不仅仅停留在聊天记录里。

## 异常处理

项目对常见异常进行了处理，包括：

- API 未配置提示
- 网络异常提示
- AI 请求失败处理
- 信息提取失败降级
- 本地数据持久化

尽量保证主聊天流程不会因增值功能失败而中断。
